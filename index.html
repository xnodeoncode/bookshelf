Bookshelf

An application written in native Javascript, HTML5 and CSS. The goal is to use native browser technologies to build a robust and responsive application. No frills, no libraries, no frameworks.

About Bookshelf

Bookshelf is a simple SPA that displays a list of books in an HTML table. Users can add, edit and delete books from the collection. Persistence The users collection of books can be stored and retrieved using HTTP Cookies, Local Storage, Session Storage, or IndexedDB.

The application was written in various implementations to provide examples of ways to create a basic client-side CRUD application using different frameworks.

The links below redirect to examples for the different implementations.

  • HTML5 and Javascript Basic HTML5, Javascript, and CSS without Persistence: Bookshelf v1
  • HTML5 and Javascript Basic HTML5, Modular Javascript Libraries, and CSS with Persistence: Bookshelf v2
  • IndexedDB Persistence: Bookshelf v3
  • Rebuilt w/ReactJS: Bookshelf v4
